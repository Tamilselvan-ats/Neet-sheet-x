# NEET 2026 Ultimate Tracker

Production-ready React + Vite app for tracking NEET syllabus progress, taking quizzes, and importing PDF-based question banks.

## Tech
- React (Vite)
- Context API
- Tailwind CSS
- LocalStorage (`neet-tracker-v1`)
- pdf.js (`pdfjs-dist`)

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Vercel
1. Push to GitHub.
2. Import repo in Vercel.
3. Framework preset: Vite.
4. Build command: `npm run build`
5. Output directory: `dist`

No backend required.
